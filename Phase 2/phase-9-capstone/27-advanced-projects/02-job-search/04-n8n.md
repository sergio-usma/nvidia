## API Server Setup

```python
# api/server.py
from flask import Flask, request, jsonify
import logging

app = Flask(__name__)
logging.basicConfig(level=logging.INFO)

# Import your modules
from scraper.aggregator import JobAggregator
from processor.job_filter import JobFilter
from processor.post_generator import LinkedInPostGenerator
from processor.categorizer import JobCategorizer

config = {
    'ollama_model': 'llama3.2',
    'ollama_host': 'http://localhost:11434'
}

@app.route('/health', methods=['GET'])
def health():
    return jsonify({'status': 'ok'})

@app.route('/api/scrape', methods=['POST'])
def scrape():
    data = request.json
    keywords = data.get('keywords', 'yacht jobs')
    location = data.get('location', 'Miami')
    
    aggregator = JobAggregator(config)
    jobs = aggregator.search_all(keywords, [location])
    aggregator.close()
    
    return jsonify({'jobs': jobs})

@app.route('/api/filter', methods=['POST'])
def filter_jobs():
    data = request.json
    jobs = data.get('jobs', [])
    
    job_filter = JobFilter(config)
    filtered = job_filter.filter_jobs(jobs)
    
    return jsonify({'filtered': filtered})

@app.route('/api/categorize', methods=['POST'])
def categorize():
    data = request.json
    jobs = data.get('jobs', [])
    
    categorizer = JobCategorizer(config)
    for job in jobs:
        job = categorizer.categorize(job)
        job = categorizer.enrich(job)
    
    return jsonify({'categorized': jobs})

@app.route('/api/generate-post', methods=['POST'])
def generate_post():
    data = request.json
    
    generator = LinkedInPostGenerator(config)
    post = generator.generate_post(data)
    
    return jsonify({'post': post})

@app.route('/api/publish-linkedin', methods=['POST'])
def publish():
    data = request.json
    post = data.get('post', '')
    
    # This would call LinkedIn API
    return jsonify({'success': True, 'post_url': 'https://linkedin.com/feed/...'})

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```

## n8n Nodes

### Custom Python Node

```python
# n8n-nodes-yacht-jobs/custom.py
from n8n_nodes_base import NodeBase

class YachtJobsNode(NodeBase):
    """Custom n8n node for yacht jobs"""
    
    def __init__(self, n8n):
        super().__init__(n8n)
    
    def execute(self):
        items = self.get_input_data()
        
        for item in items:
            # Process item
            result = self.process_job(item)
            self.add_output_item(result)
    
    def process_job(self, item):
        # Your processing logic
        return item
```

## Automation Rules

### Scheduled Scanning

- Every 6 hours: Scrape new jobs
- Daily at 8 AM: Process and post to LinkedIn
- Weekly: Generate job market report

### Filtering Rules

- Minimum salary: $50,000/year
- Exclude: Intern, Unpaid, Trainee
- Categories: All yacht categories

### Posting Rules

- Maximum 3 posts per day
- No duplicate posts
- Include link to original job
